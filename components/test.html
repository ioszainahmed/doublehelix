Component Architecture

  • LogoImage.js - Renders logo images (Single Responsibility)
  • LogoLabel.js - Renders text labels (Single Responsibility)
  • LogoItem.js - Combines image + label (Interface Segregation)
  • LogoData.js - Stores logo configuration (Liskov Substitution)
  • InfiniteLogoCarousel.js - Orchestrates carousel (Dependency Inversion)
